# AI Chat Bot 
公開中URL：https://chat-bot-app-v1-fd1f8c178077.herokuapp.com/

## 概要
Djangoで構築したAIチャットサポートのテンプレートです。
ユーザーの利便性とプライバシーを考慮し、最終アクセスから24時間経過で自動消去される会話履歴保存機能を搭載しています。


## 主な機能
### 機能詳細

* **リアルタイムチャットUI**: Bootstrap 5を使用したレスポンシブデザイン。PC・スマホ両対応。
* **会話履歴の永続化**: ブラウザの `LocalStorage` を使用し、ページを閉じても会話が継続。
* **24時間自動クリア**: セキュリティ対策として、最後の会話から24時間経過した履歴は自動的に削除されます。
* **UX最適化**: AIの回答待ち時間に「読み込み中...」インジケーターを表示し、スムーズな書き換えを実現。

### 履歴管理
* 直近20件までのメッセージを保存。

### レスポンシブ設計
* スマホ閲覧時にはチャットログの高さを画面に合わせて最適化。

## 使用技術
- **HTML5 / CSS3**: チャットバブルのカスタマイズおよびレイアウト。
- **JavaScript (ES6+)**: 
  - `Fetch API` による非同期通信。
  - `LocalStorage` を活用した状態保持および24時間有効期限管理ロジック。
- **Bootstrap 5.3.0**: モバイルファーストのレスポンシブデザインおよびUIコンポーネント。
- **Python 3.11.9**: メインロジックの実装。
- **Django 4.2.2**: 
  - テンプレートエンジンを用いたフロントエンドとの統合。
  - CSRF保護機能を利用した安全なPOST通信。
- **URLSearchParams**: フォームデータのシリアライズ処理。
- **JSON**: ブラウザ・サーバー間のデータ受け渡し。
